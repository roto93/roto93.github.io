Outline

1. 創建 github 帳號，並取得網域名稱

首先到 github 官網註冊帳號，完成後登入
接著點擊總上繳的 new 來新增一個 repository

然後在 repository name 的地方輸入 你的帳號名.github.io 。注意這是唯一的名字，打任何其他的都不對。是說我也不知道會有什麼問題，可能跟是不是根目錄有關吧。
